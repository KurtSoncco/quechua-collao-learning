**Yachasun Quechua** is an interactive web platform designed to help users master the top **150 Quechua Collao verbs**. Through a variety of exercises, users can learn verb meanings, practice complex conjugations across multiple tenses, and translate contextual paragraphs.

## ✨ Features

- **Top 150 Verbs:** A curated database of essential Quechua Collao verbs with roots and meanings.
- **Interactive Matching:** Pair Quechua infinitives with their English/Spanish translations.
- **Advanced Conjugation:** Practice 7 persons across 8 tenses:
  - Present, Past, Future, Imperative, Conditional
  - Present Progressive, Past Progressive, Future Progressive
- **Contextual Translation:** Work through multi-sentence paragraphs to apply verb knowledge in real scenarios.
- **Reference & Study Guide:** A built-in vocabulary search and grammar table for quick review.
- **Instant Feedback:** Get scored on every exercise and review the correct answers to learn from mistakes.

## 💻 Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/KurtSoncco/quechua-collao-learning.git
   cd quechua-collao-learning
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

---

## 📖 How to Use

1. **Review:** Start with the "Review Verbs & Grammar" section to study the 150 verbs and suffix rules.
2. **Practice:** Click "Start Learning" to begin the exercise sequence.
3. **Score:** Complete all three sections (Matching, Conjugation, Translation) to see your final performance.
4. **Repeat:** The verbs and exercises randomize every time you restart!
